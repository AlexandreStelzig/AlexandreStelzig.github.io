Randomizer App

Description

Having a hard time chosing between 2 or more things? Well, I did until I created the "Randomizer" application on Android. This application allows the user to create lists with items that can be randomized in a random order. Each item also has an associated weight which changes the odds of it being picked.

Media


Main Features

  • Create Lists from the application's drawer
  • Add items (name and random weight)
  • Set an item active or inactive
  • Sort the item's position in the list with a dragview listview
  • Long click the item to edit or delete it
  • Edit the list's name or delete it by clicking the menu options
  • Randomize the list and display the results in a custom listview dialog (with the option to re-randomize it)

Line tracer robot

Description

During my student exchange in South Korea, I had the opportunity to take a Introduction to Robot programming class. The final project of this class was to build a Line Tracer robot using an Arduino.

Media


Main Features

  • Robot using Arduino and 4 light sensors
  • Turn on/off button located on the top of the robot
  • Automatic following as well as steering of straight and curved lines
  • Stopping logic when all sensors are on a dark line
  • Result of race: succesfully finished the track above in 34 seconds
