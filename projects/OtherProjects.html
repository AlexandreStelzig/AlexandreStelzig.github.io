Randomizer App

Description

Having a hard time chosing between 2 or more things? Well, I did until I created the "Randomizer" application on Android. This application allows the user to create lists with items that can be randomized in a random order. Each item also has an associated weight which changes the odds of it being picked.

Media


Main Features

  • Create Lists from the application's drawer
  • Add items (name and random weight)
  • Set an item active or inactive
  • Sort the item's position in the list with a dragview listview
  • Long click the item to edit or delete it
  • Edit the list's name or delete it by clicking the menu options
  • Randomize the list and display the results in a custom listview dialog (with the option to re-randomize it)

Video Game Store App

Description

During the Analysis and Design of User Interfaces project at the University of Ottawa, me and my friend decided to create a fictional android application. The goal of this project was to create an user friendly experience of a mobile shop using hardcoded games/consoles as well as loading images from the web using AsyncResponses and urls.

Media


Main Features

Home Page

  • Randomily selects console/games which as a 'deal'
  • Option to view all consoles or games

Shop

  • Shop by consoles, games by category or games by console
  • Filter results by console and/or categories
  • Search product by name
  • Sort by Name (A-Z / Z-A) or Price (Low-High / High - Low)
  • Short description includes: Main picture, Title, console type, ESRB, published date and price
  • Open product's description by clicking it or add it to the cart

Product's description page

  • Multiple pictures of the product
  • Long description as well as product's title, ESRB rating, console type, publisher, date published and price
  • Option to add product to wishlist or to the cart

Cart

  • Display all the products in the cart
  • If not logged in, the user is redirected to the sign in page
  • Option to add/remove copies of the product
  • Total price without taxes displayed at the bottom

Checkout

  • Display all the products to be checkout
  • Option to add/remove copies of the product
  • Choice between two shipping methods: Standar and Expedited (+25$ on total)
  • User can add or choose the payment method and shipping address
  • Total cost displayed with the total cost (including taxes and shipping)

Order confirmation

  • Information of the product bought
  • Order, Shipping, payment and cost information

Account

  • Create an account with an email, password and first/last name
  • Login using the email and password
  • My orders: information of all previous orders
  • Payment method: add/edit/delete payment methods
  • Account information: edit email, name and password
  • Addresses: add/edit/delete payment methods
  • WhishList: display products in the user's whichlist (can be removed by the user or automatically when buying the product)

Line tracer robot

Description

During my student exchange in South Korea, I had the opportunity to take a Introduction to Robot programming class. The final project of this class was to build a Line Tracer robot using an Arduino.

Media


Main Features

  • Robot using Arduino and 4 light sensors
  • Turn on/off button located on the top of the robot
  • Automatic following as well as steering of straight and curved lines
  • Stopping logic when all sensors are on a dark line
  • Result of race: succesfully finished the track above in 34 seconds
