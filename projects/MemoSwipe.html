MemoSwipe

"MemoSwipe" is a flash card application that serves the purposes of learning new languages faster by engaging the user with repetitive memorization. I created this application when I started learning Korean and wanted to have a better way to learn vocabulary. When I tried existing flash cards applications, there was always a few features that I wanted that was missing, so I told myself "why not build one instead?" Since then, this application helped me greatly while learning new languages.
