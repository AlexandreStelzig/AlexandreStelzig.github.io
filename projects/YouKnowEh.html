You Know Eh?

Description

"You Know Eh?" is a flash card application that serves the purposes of learning new languages faster by engaging the user with repetitive memorization. I created this application when I started learning Korean and wanted to have a better way to learn vocabulary. When I tried existing flash cards applications, there was always a few features that I wanted that was missing, so I told myself "why not build one instead?" Since then, this application helped me greatly while learning new languages.

Main Features

Review

  • Review cards by selecting a deck in a random or chronological order
  • Review by 'Language Learning' -> 'Native Language' or vice-versa
  • Toggle a card from review for X hours (also includes an show preview and undo functions if misclicked)
  • Settings for review includes: Modify current card's decks, view card detailed information, shuffle order, always show answers

Card List

  • Display all cards or cards for a selected deck
  • Search card(s) by native or learning name
  • Clicking on a card opens the card's information
  • Long clicking or clicking on more gives the option to: edit card, open quick info, open quick edit, delete card
  • Sort cards by: Language Learning (A-Z) or (Z-A), Native Language (A-Z) or (Z-A), Date Created (New-Old) or (Old-New), Date Modified (New-Old) or (Old-New), Number of decks
  • Add or Remove cards for current selected deck
  • Toggle cards from review for current selected deck
  • Import cards from a csv file
  • Reverse the display information text location
  • Filter the review and non-review cards
  • Quick create (no comments)
  • Open card creation card (if open with a deck selected, the deck will be automaticly added to the card's deck list)

Card Information

  • Language Learning question
  • Meaning in Native Language
  • Comments
  • Card's Decks information (can be in 0, 1 or multiple decks)
  • Date Created and Modified information

Deck List

  • Display all decks in the order the user placed them
  • Clicking on a deck opens the card list on the selected deck
  • Long clicking or clicking on more gives the option to: edit deck, quick information, export deck (csv file), delete deck
  • Reorder decks positions
  • Import one (csv) or multiple (zip) deck(s)
  • Export all decks (zip file)
  • Create new deck

Quiz

  • 3 types of quizes: Reading, Writing and Multiple Choice
  • Reading: Similar to review but with failed/pass button instead of show next
  • Writing: Spell correctly the given word into the targeted language
  • Multiple Choice: Chose the right translation between 4 multiple choice options
  • New Quiz option: quiz type, include non-review cards, orientation of question asked, which deck(s) to use
  • Take a break and come back to the quiz anytime with the save and continue function

Quiz History

  • Display the past quizzes information and results (# pass/failed) and time taken to complete it
  • Export/Import (csv) quizzes history

Statistics

  • Display the statistics of: Number of cards added or Total number of cards for a date range
  • Select one, multiple or all decks statistics

Profile

  • Multiple profiles with unique cards and decks
  • Each profile has: a name, labels for the front and back of the card, settings, 8 types of background color, background image...
  • Settings include: Default sorting, Quick toggle time (hours), Merge Duplicate cards, View deleted cards, Export profile(s) and more...

Profile List

  • Scroll list with dynamic background changing depending of the selected profile
  • Display last time that the profile was opened
  • Create new profile
