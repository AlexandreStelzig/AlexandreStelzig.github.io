Goal Tracker

Description

One of the hardest thing I think for most human beings is to keep themselves motivated and productive everday. To combat this issue, I decided to create 'Goal Tracker' an android application that keeps track of daily and weekly goes that are set by the user. While the goals aren't accomplished, the application send reminders to the user on his phone. Also, by keeping track of everyday's progress, it makes the user more productive by not wanting to leave a goal unaccmplished.

Media


Main Features

Goal Page

  • Main page of the application where daily and weekly goals are displayed for the selected date
  • Set the accomplish state of a goal
  • Navigation arrows for daily and weekly goals as well as a calendar picker to select a different date
  • Filter displayed goals by category (saves the state for next time the app opens)

Categories

  • Create/Add/Edit categories with an unique name
  • Fully customizable color

Goal Management

  • Create/Edit/Delete goal with a name, number repeat, counter, category and mandatory state
  • 3 types of counters: time(s), minute(s) and hour(s)
  • Create a goal from an existing rule
  • Filter displayed goals by category or by active state

Rule Management

  • Create/Edit/Delete rule with a name, number repeat, counter, category and mandatory state
  • If the rule is daily, the repeat rule can be: tomorrow only or any days of the week (e.g. every Friday or every Monday and Tuesday)
  • If the rule is weekly, the repeat rule can be: next week only or any # of the week (e.g. every week or every 2 weeks)
  • Filter displayed rules by category or by active state

Statistics

  • Display the statistics for: pourcentage accomplished, or number of goals
  • Clicking on a deck opens the card list on the selected deck
  • Long clicking or clicking on more gives the option to: edit deck, quick information, export deck (csv file), delete deck
  • Reorder decks positions
  • Import one (csv) or multiple (zip) deck(s)
  • Export all decks (zip file)
  • Create new deck

Comments

  • 3 types of quizes: Reading, Writing and Multiple Choice
  • Reading: Similar to review but with failed/pass button instead of show next
  • Writing: Spell correctly the given word into the targeted language
  • Multiple Choice: Chose the right translation between 4 multiple choice options
  • New Quiz option: quiz type, include non-review cards, orientation of question asked, which deck(s) to use
  • Take a break and come back to the quiz anytime with the save and continue function

Statistics

  • View statistics for Daily goals, Weekly goals or specific goals for a date range
  • Display the statistics of Number of cards added or Total number of cards for a date range

Settings

  • 8+ settings to customize the looks and functionality of the application
  • Automatic backup system
  • Export and Import function to recover data lost or when changing phones using CSV files

Notifications

  • Customizable notification with a start, end and interval time
